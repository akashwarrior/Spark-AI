# Spark AI 🔥

Spark AI is an AI-powered website builder that helps you create beautiful, production-ready websites directly in the browser. It integrates productivity tools and streamlines the entire development workflow — from layout to code — with the help of Gemini AI.

## 🚀 Features

- AI-assisted component generation
- Real-time code editing and preview
- Minimal, high-performance UI
- Browser-based — no installation needed

## 🧪 Tech Stack

- Next.js (App Router)
- TypeScript
- Gemini AI API
- Tailwind CSS
- React
